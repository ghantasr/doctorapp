# How to Set Follow-Up Date for Patients

## Location
You can set the follow-up date for a patient from the **Patient Detail Screen**.

## Steps to Set Follow-Up Date:

### 1. Navigate to Patient
- Go to **Patients** tab in the bottom navigation
- Search or select a patient from the list
- Tap on the patient to open their detail screen

### 2. Locate Follow-Up Section
- Scroll to the **Assignment Status** card (top of the screen)
- Find the **Follow-Up Date** section with a calendar icon

### 3. Set the Date
- Click the **calendar icon** (+ icon if no date is set, edit icon if date exists)
- A dialog will open showing:
  - Current/suggested follow-up date
  - Date picker to change the date
  - Note: "Patient will receive a reminder 2 days before this date"

### 4. Select Date
- Tap on the date card to open the date picker
- Choose the desired follow-up date
- The date must be in the future (up to 1 year ahead)

### 5. Save
- Click **Save** to confirm
- Success message will appear
- Click **Clear** if you want to remove the follow-up date

### 6. View Follow-Up Patients
- Open the **drawer menu** (hamburger icon)
- Select **Follow-Up Patients**
- See all patients categorized by:
  - **Overdue** (red) - Missed their follow-up date
  - **Today** (blue) - Due today
  - **Upcoming** (green) - Future follow-ups
- Patients within 2 days will show an orange notification indicator

## Visual Indicators

### In Patient Detail:
- **Blue calendar icon** - Follow-up date is set
- **Gray text "Not set"** - No follow-up date
- **Edit icon** - Change existing date
- **Plus icon** - Set new date

### In Follow-Up Patients List:
- **Red cards** - Overdue follow-ups (need immediate attention)
- **Blue cards** - Today's follow-ups
- **Green cards** - Upcoming follow-ups
- **Orange dot** - Reminder notification sent (within 2 days)

## Notifications
The system is configured to:
1. Send reminder to patient **2 days before** the follow-up date
2. Prompt patient to book an appointment
3. Show the patient in your "Follow-Up Patients" list

## Default Behavior
- Default follow-up date suggestion: **7 days from today**
- Can be changed to any future date
- Can be cleared if no longer needed
- Patient disappears from follow-up list once date is cleared

## Use Cases

### After Treatment Completion:
1. Complete the treatment for the patient
2. Set a follow-up date (e.g., 2 weeks for check-up)
3. Patient gets reminder 2 days before
4. Patient books appointment
5. You see them in "Today" section on follow-up date

### For Ongoing Care:
1. Set initial follow-up date
2. After each visit, update to next follow-up date
3. Track all follow-ups in one place
4. Never miss a patient check-up

## Tips
- **Set realistic dates** - Give enough time for healing/recovery
- **Use the Follow-Up Patients view** - Check daily for upcoming appointments
- **Clear dates** - If patient no longer needs follow-up
- **Update dates** - If patient reschedules or needs earlier/later follow-up

## Quick Access
**Patient Detail → Assignment Status Card → Follow-Up Date → Calendar Icon**

Or

**Drawer Menu → Follow-Up Patients → See All Scheduled**
